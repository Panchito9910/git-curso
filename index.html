Ejemplo de como funciona git

  1. Inicializar el repositorio del código con "git init"
  2. "git status" se usa para ver que archivos estamos trabajando en el proyecto
  3. "git add (nombreDelArchivo)" se usa para añadir un archivo al proyecto
  4. "git commit" se usa para hacer el snapshot del proyecto
  5. "git log" se usa para ver los commits que se han realizado
  6. "git checkout --" se usa para revertir los cambios de los archivos
  7. "git diff (nombreDelArchivo)" se usa para ver las diferencias entre los archivos modificados
  8. ".gitignore" es el nombre que debe llevar un archivo para que git ignore los archivos que no se desean
    agregar al proyecto. Este archivo se tiene que añadir al proyecto.
